About mesh preferences

In the MESH module you can set mesh preferences, default settings, which can be used in later sessions with this module. To set preferences, select in the main window: Preferences / Mesh (other preferences are described in the KERNEL module)

  • Display mode allows to set Wireframe, Shading, Nodes or Shrink presentation mode as default.  This functionality is described in full detail in other chapter.

  • Colors and Size settings are also described in other chapter. By setting them here you will them in all later sessions.

  • Scalar Bar allows you to set Scalar Bar Preferences

      

In the picture above  the default settings are shown

  • In the menu Preferences - Selection you can set default colors line width for displaying Mesh Objects, Mesh Elements and Preselection. You can also set the precision level for Nodes and Elements.  

      

  • Via Quality Controls / Precision it is possible to set the Number of digits after point for numbers in Quality Control diagrams or present integer values (by default)

      

  • Finally by toggling Quality Controls / Display entity you will display by default both faces and edges of an object.
